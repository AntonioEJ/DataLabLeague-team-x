# Schemas Contracts

Contratos de esquemas de datos. Cada archivo `[entidad].schema-contract.md` define la estructura, tipos, restricciones y versión del esquema de una entidad de datos.

**Propósito**: Garantizar compatibilidad y evolución controlada de esquemas entre sistemas.

**Template**: Ver [../templates/data-contract-template.md](../templates/data-contract-template.md)

**Índice**: Ver [../contracts.md](../contracts.md)
