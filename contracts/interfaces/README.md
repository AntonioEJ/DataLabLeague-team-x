# Interface Contracts

Contratos de interfaces entre sistemas proveedores y consumidores. Cada archivo `[origen]-to-[destino].interface-contract.md` define el acuerdo formal de entrega: formato, esquema, frecuencia, validaciones y manejo de errores.
