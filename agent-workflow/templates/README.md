# Templates — DataLab League Agent Workflow

## Propósito

Plantillas reutilizables para los artefactos de la cadena de agentes.

## Archivos

| Template | Uso |
|---|---|
| `agent-input-template.json` | Base para crear nuevos artefactos de input |
| `agent-output-template.json` | Base para crear nuevos artefactos de output |
| `handoff-template.json` | Base para crear nuevos artefactos de handoff |
| `governance-assessment-template.md` | Template de evaluación de gobierno (Agente 02) |
| `crisp-dm-phase-template.md` | Template para documentar una fase CRISP-DM |
| `evidence-template.md` | Template para registrar evidencia |
| `decision-record-template.md` | Template para registrar decisiones (DEC-XXX) |
| `open-question-template.md` | Template para registrar preguntas abiertas (OQ-XXX) |
| `risk-template.md` | Template para registrar riesgos (RSK-XXX) |

## Cómo Usar

1. Copiar el template al destino correcto.
2. Reemplazar todos los valores `{{placeholder}}` con datos reales.
3. Actualizar `status` de `draft` a `in_review` cuando esté listo para revisión.
