# Prompt: Final Evidence

## Objetivo
Compilar evidencia final del proyecto.

## Instrucciones
1. Recopilar artefactos
2. Documentar procesos
3. Generar reportes
4. Crear demostración
5. Compilar evidencia

## Entregables
- Evidencia compilada
- Reportes finales
- Demo y presentación

## Criterios de Aceptación
- Evidencia completa
- Reportes claros
- Demo funcional
