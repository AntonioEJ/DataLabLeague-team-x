# Prompt: Data Quality Rules

## Objetivo
Crear y validar reglas de calidad de datos.

## Instrucciones
1. Analizar fuentes de datos
2. Definir reglas de validación
3. Implementar validaciones
4. Crear reportes
5. Monitorear calidad

## Entregables
- Reglas de calidad documentadas
- Validaciones implementadas
- Reportes de monitoreo

## Criterios de Aceptación
- Reglas claramente definidas
- Validaciones automáticas
- Reportes periódicos
