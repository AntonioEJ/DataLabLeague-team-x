# Fuentes de Datos

Catálogo de fuentes de datos disponibles para el producto.

## Formato

```
### DS-XXX: [Nombre de la fuente]
- **Sistema origen**: [DB, API, archivo, etc.]
- **Tabla / endpoint**: [Nombre exacto]
- **Owner técnico**: [Equipo o persona]
- **Data Owner**: [Responsable de negocio]
- **Frecuencia de actualización**: [tiempo real | diaria | semanal]
- **Volumen aproximado**: [filas/día o GB]
- **Contiene PII**: sí | no
- **Datos sensibles**: sí | no
- **Clasificación**: público | interno | confidencial | restringido
- **Data contract**: sí | no | pendiente
- **Calidad conocida**: [Observaciones]
```

_Fuentes de datos pendientes de inventario en pasos 01 y 02._
