# 00-shared — Contexto Compartido

Esta carpeta contiene el contexto compartido por **todos los agentes** de la cadena.

## Regla de Uso

Antes de iniciar cualquier tarea, cada agente **debe leer** los archivos de esta carpeta.  
Si el agente identifica nueva información relevante, **debe actualizar** el archivo correspondiente.

## Archivos

| Archivo | Propósito |
|---|---|
| `context.md` | Descripción del producto de datos, negocio y contexto |
| `assumptions.md` | Supuestos validados por los agentes |
| `open-questions.md` | Preguntas abiertas y su estado de resolución |
| `decisions.md` | Decisiones arquitectónicas y de negocio tomadas |
| `glossary.md` | Glosario de términos del dominio |
| `business-kpis.md` | KPIs de negocio que el producto debe medir |
| `data-sources.md` | Fuentes de datos disponibles y sus características |
| `risk-register.md` | Registro de riesgos identificados |
