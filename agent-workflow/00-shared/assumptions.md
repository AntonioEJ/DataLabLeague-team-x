# Supuestos

Todos los supuestos identificados durante el workflow.  
Si un supuesto resulta falso, convertirlo en pregunta abierta bloqueante.

## Formato

```
### ASM-XXX: [Título corto]
- **Descripción**: [Qué se asume]
- **Origen**: [Agente o persona que lo identificó]
- **Fecha**: YYYY-MM-DD
- **Impacto si es falso**: [Qué pasaría]
- **Estado**: activo | invalidado
```
