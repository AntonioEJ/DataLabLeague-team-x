# Preguntas Abiertas

Registro de preguntas sin respuesta. Toda pregunta bloqueante impide el handoff del agente correspondiente.

## Formato

```
### OQ-XXX: [Título corto]
- **Pregunta**: [Pregunta específica]
- **Origen**: [Agente o persona]
- **Fecha**: YYYY-MM-DD
- **Prioridad**: high | medium | low
- **Bloquea al siguiente agente**: sí | no
- **Agente bloqueado**: [nombre si aplica]
- **Resolución**: [Respuesta cuando esté disponible]
- **Resuelta por**: [Persona o agente]
- **Fecha resolución**: [YYYY-MM-DD]
- **Estado**: abierta | resuelta | cancelada
```
