# Inputs — Agente 01: Enrich Data Story User

## Qué se Guarda Aquí

Documentos de entrada del requerimiento inicial de negocio.

## Archivos

| Archivo | Descripción | Obligatorio |
|---|---|---|
| `initial-requirement.md` | Texto del requerimiento inicial de negocio | Sí |
| `source-documents.md` | Referencias a documentos fuente (PDFs, specs, emails) | No |

## Naming Convention

- Archivos en `kebab-case.md`
- Si hay múltiples versiones: `initial-requirement-v2.md`

## Reglas de Revisión

- Estos archivos son de **solo lectura** para los agentes de la cadena.
- Solo el solicitante de negocio puede modificar `initial-requirement.md`.
- Si el requerimiento cambia después de que Agente 01 ya inició, registrar en `00-shared/open-questions.md`.

## Relación con PR

- Los archivos de input deben estar presentes en el PR de inicio del ciclo.
