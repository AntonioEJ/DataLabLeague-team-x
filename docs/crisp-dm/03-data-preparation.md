# Fase 3: Preparación de Datos (Data Preparation)

## Objetivo
Construir el conjunto de datos final para modelado o análisis.

## Tareas

### 3.1 Seleccionar Datos
- Elegir registros
- Elegir campos relevantes
- Justificar exclusiones
- Documentar decisiones

### 3.2 Limpiar Datos
- Manejar valores faltantes
- Eliminar duplicados
- Corregir inconsistencias
- Validar tipos

### 3.3 Construir Variables
- Crear nuevas variables
- Transformar variables
- Normalizar/Escalar
- Codificar categorías

### 3.4 Integrar Datos
- Combinar fuentes
- Reconciliar
- Validar joins
- Resolver conflictos

### 3.5 Formatear Datos
- Formato final
- Estructura
- Documentación
- Exportar datasets

## Entregables
- Dataset preparado
- Scripts de preparación
- Documentación de transformaciones
- Validaciones de calidad
