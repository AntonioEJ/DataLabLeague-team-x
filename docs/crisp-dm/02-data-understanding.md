# Fase 2: Entendimiento de Datos (Data Understanding)

## Objetivo
Recopilar datos iniciales y aumentar el entendimiento de los datos disponibles.

## Tareas

### 2.1 Recopilar Datos Iniciales
- Identificar fuentes
- Recopilar muestras
- Documentar estructura
- Validar acceso

### 2.2 Describir Datos
- Cantidad de registros
- Campos disponibles
- Tipos de datos
- Estadísticas básicas

### 2.3 Explorar Datos
- Distribuciones
- Relaciones
- Anomalías
- Patrones

### 2.4 Verificar Calidad
- Valores faltantes
- Duplicados
- Inconsistencias
- Outliers

## Entregables
- Reporte exploratorio
- Diccionario de datos
- Evaluación de calidad
- Plan de preparación
