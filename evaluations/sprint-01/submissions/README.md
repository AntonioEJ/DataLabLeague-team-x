# Submissions — Sprint 01

Aquí cada equipo sube sus entregables del Sprint 1 de forma manual.

## Instrucciones para los equipos

1. Subir los archivos en la carpeta correspondiente a su equipo (`team-01/`, `team-02/`, `team-03/`).
2. El archivo de instrucciones para GitHub Copilot debe llamarse `copilot-instructions.md`.
3. El README debe llamarse `README.md`.
4. Hacer commit en la rama `submissions/sprint-01/team-XX` y abrir un PR.

## Estructura esperada por equipo

```
submissions/
└── team-XX/
    ├── README.md                  ← README general del repositorio del equipo
    └── copilot-instructions.md   ← instrucciones para GitHub Copilot
```
